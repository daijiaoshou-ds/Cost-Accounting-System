# 归因分析——用户问"为什么"时你怎么答

当用户在 Phase 5 追问细节时，翻到这里找对应的分析路径。

## 你手上有这些牌

| 想看什么 | 文件在哪 | 里面有啥 |
|---------|--------|---------|
| 总览 | `output/_index.json` | 算了几个月、矩阵多大、多少物料、总成本 |
| 波动摘要 | `output/cost_fluctuation.json` | Top N 产品年汇总、月度单价、波动排名 |
| 波动明细 | `output/cost_fluctuation.csv` | 每个产品×每个月的 完工金额/数量/单价 |
| 毛利摘要 | `output/margin_analysis.json` | 总毛利、月度/产品线毛利率、异常批次 |
| 毛利明细 | `output/margin_analysis.csv` | 每批次的 收入/成本/毛利 |
| 校验日志 | `output/_pipeline_log.json` | 校验结果、错误/警告 |
| 收发存 | `output/{month}/收发存汇总.csv` | 每个物料当月的收发存 |
| 节点成本明细 | `output/{month}/成本明细.csv` | 每个节点的总成本（材料/人工/制费），含全节点 X 矩阵值 |
| 工单成本拆分 | `output/{month}/工单明细.csv` | 每个工单的完工材料费/人工/制费，可按产品汇总 |
| 销售成本拆分 | `output/{month}/销售成本明细.csv` | 每批次的销售成本（材料/人工/制费）及销售金额 |

## 路径 A：用户问"这个产品的成本为什么波动了？"

1. 打开 `cost_fluctuation.json`，在 `fluctuation_ranking` 里找到这个产品。
2. 看一眼：月均单价多少、最高/最低单价差多少、最大波动发生在哪个月。
3. 打开 `cost_fluctuation.csv`，看这个产品各月的完工数量和完工单价。
4. 判断波动类型：
   - **单价在变、数量稳定** → 采购价格波动或 BOM 结构变了
   - **数量在变、单价稳定** → 产量波动导致单位分摊变化
   - **都在变** → 综合影响
5. 要深挖？打开波动最大的两个月对应的 `成本明细.csv` 和 `工单明细.csv`，筛选这个产品——看看是材料费变了、人工变了、还是制费变了。
6. 回复用户：哪个产品、哪两个月之间、单价从多少变到多少、波动多少、主要是什么原因、有什么建议。

回复模板：
```
《产品编码》在《月份A》到《月份B》期间，
完工单价从 ¥《A》变动到 ¥《B》，波动《百分比》。
主要原因是《归因说明》。
建议：《建议》。
```

## 路径 B：用户问"这个批次为什么毛利这么低 / 亏了？"

1. 打开 `margin_analysis.json`，看 `anomalies`（异常批次列表）。
2. 打开 `margin_analysis.csv`，找到这个批次，确认收入和成本各是多少。
3. 先判断：
   - **收入为 0 或异常低** → 告诉用户"销售金额字段可能没填或填错了，没有收入数据的话毛利率算不出来。"
   - **成本异常高** → 继续往下查。
4. 打开这个批次对应月份的 `收发存汇总.csv`，看该产品当月的收发存情况。
5. 打开这个月份对应产品的 `工单明细.csv`，筛选该产品，看完工材料费/人工/制费的构成——哪个占比异常。
6. 想看销售端的成本拆分？打开 `销售成本明细.csv`，能看到这个批次的 销售成本_料/工/费 各自占了多少。
7. 回复用户：哪个批次、毛利率多少、成本高在哪个维度、可能的原因、建议。

---

## 路径 C：用户问"全年/这几个月的成本趋势怎么样？"

1. 打开 `cost_fluctuation.json`：
   - 看 `fluctuation_ranking`：波动最大的前几名是谁
   - 看 `fluctuation_distribution`：大幅/小幅/持平各占多少
   - 看 `month_pair_statistics`：逐月上涨/下跌产品数变化
2. 找到波动最大的月份和波动最小的月份。
3. 找到全年最稳定的产品（波动接近 0 的）和最不稳定的产品（波动最大的）。
4. 如果有毛利率数据，打开 `margin_analysis.json` 看月度毛利率趋势——是稳定、上升、还是下降。
5. 可以看看 `_pipeline_log.json` 里 `stage_times` 各月的矩阵维度——数据量是不是逐月增长的。
6. 回复用户：整体趋势是涨还是跌、关键转折在哪个月、哪些产品是定海神针哪些是波动王、有没有需要警惕的信号。
